# YouTube Downloader GUI

A modern YouTube video downloader built with Python, CustomTkinter, and yt-dlp.

## Features

- Modern GUI
- Dark Mode UI
- Download YouTube videos
- Download audio only (MP3/Audio)
- Quality selection
- Download progress bar
- Download speed display
- Remaining time display
- Select download folder
- Multithreaded download (No UI freezing)

# Installation

## 1. Clone the repository

git clone https://github.com/YOUR_USERNAME/youtube-downloader-gui.git

## 2. Enter project folder

cd youtube-downloader-gui

## 3. Install dependencies

pip install -r requirements.txt

---

# Required Libraries

pip install yt-dlp customtkinter pillow

---

# Run the Application

python youtube_downloader.py

---

# Build EXE

Install PyInstaller:

pip install pyinstaller

Build executable:

pyinstaller --onefile --windowed youtube_downloader.py

The EXE file will be created inside:

dist/

---

# FFmpeg

Download FFmpeg:
https://ffmpeg.org/download.html

Place ffmpeg.exe next to the executable file.
